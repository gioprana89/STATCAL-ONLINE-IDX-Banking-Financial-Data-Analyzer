# STATCAL ONLINE IDX Banking Financial Data Analyzer with Regression

This Streamlit application explores IDX banking financial data through filters, descriptive statistics, grouped descriptive statistics, correlation matrix, multi-panel line charts, scatterplots, multiple linear regression, and classical assumption tests.

## Main features
- Data filtering by Year and Ticker Code
- Univariate descriptive statistics
- Grouped descriptive statistics
- Correlation matrix and heatmap
- Multi-panel mean line chart
- Scatterplot panel
- Multiple linear regression
- Normality tests, VIF/tolerance, heteroskedasticity tests, autocorrelation tests, outlier/influence diagnostics
- PNG export with default 1200 DPI
- Excel export for tables and regression outputs

## Run
```bash
pip install -r requirements.txt
python -m streamlit run main.py
```
